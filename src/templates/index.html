SilverBullet

SilverBullet is a live, distributed analysis platform for high-frequency trading applications. For details about the project and source code, see the Github repo.

The current version provides live optimization of a cryptocurrency portfolio according to Modern Portfolio Theory, using a Monte Carlo strategy.

Dashboard

Dashboard - provides a visualization of optimal portfolios, latency and some additional statistics.

API Endpoints

Latest optimal portfolios, and other computation results, are also available through the REST API:
  • Individual asset statistics
  • Individual portfolio statistics
